# ![Noodle Man RPG](legacy/logosmall.png)

![CC BY-NC-SA Shield](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)

This is a reproduction of a game originally developed in 2005 by a teenager learning programming. It has been preserved close to its original state with spelling errors and bugs. AI has been used to assist in converting the code from PHP to JavaScript. 

The game can be played at [nmrpg.mattw.art](https://nmrpg.mattw.art/) or downloaded from [the Releases page](releases). To run the downloaded version open legacy/index.html in your browser.
